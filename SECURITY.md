# Security Policy

## Supported Versions

The latest released version of Abyss receives fixes. Older tagged releases are
not maintained.

## Reporting a Vulnerability

This is a Garmin Connect IQ watch face. It requests **no permissions**, reads only
on-device sensor/activity data (battery, steps, barometric altitude, weather, heart
rate), and does not transmit any data off the device.

If you find a security or privacy issue, please report it privately rather than
opening a public issue:

- Use GitHub's [private vulnerability reporting](../../security/advisories/new), or
- Email **infantrykiller@gmail.com** with the details and steps to reproduce.

Please allow up to 7 days for an initial response. Thanks for helping keep the
project safe.
